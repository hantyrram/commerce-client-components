Properties
Name Type Attributes Default Description
entity Entity The Entity class that this component will work on.
entityPrimaryKey string The primary key to be used in routing, does not have to be the primary key of the entity on the database as long as the value is unique.
reader null | true | Object If adder is null or true, this indicates reader is enabled but lets Bread use the default reader. If reader is a non null Object, it will use it as the reader ui for the Entity.
adder null | true | Object <optional>
 If adder is null or true, this indicates adder is enabled but lets Bread use the default adder. If adder is a non null Object, it will use it as the adder ui for the Entity.
editor null | true | Object <optional>
 If editor is null or true, this indicates editor is enabled but lets Bread use the default editor. If editor is a non null Object, it will use it as the editor for the Entity.
deleter boolean <optional>
 false Enables the delete action on if true.
